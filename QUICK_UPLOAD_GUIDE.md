# 🚀 Quick GitHub Upload Guide

## ⚡ FASTEST WAY TO UPLOAD (2 Minutes)

### Option 1: Direct Web Upload (Recommended)
1. **Open**: https://github.com/mahifeysal69-lang/noor-al-imaan
2. **Click**: "Add file" → "Upload files"
3. **Drag & Drop**: All files from your project folder
4. **Commit**: "Initial commit - Noor Al Imaan Registration System"
5. **Click**: "Commit changes"

### Option 2: Automated Script
1. **Get GitHub Token**: 
   - Go to https://github.com/settings/tokens
   - Click "Generate new token (classic)"
   - Check "repo" permissions
   - Copy the token
2. **Update Script**: 
   - Open `upload-to-github.js`
   - Replace `YOUR_GITHUB_PERSONAL_ACCESS_TOKEN` with your token
3. **Run**: `node upload-to-github.js`
